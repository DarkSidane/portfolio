Web Sémantique

Le web sémantique consiste à rendre compréhensible les données sur le web par des machines. Le terme a été proposé par Tim Berners Lee en 2001 pour désigner une évolution du web qui permettrai au données d'être utilisables et interprétables automatiquement par des logiciels. Pour permettre cette évolution, des standard W3C ont été développés.

Durant ma formation d'ingénieur j'ai eu une approche du web sémantique dans laquelle j'ai pu voir les différents outils permettant de structuré les données (XML, XPath, XML Schema, XSLT), de faire des relations entre ces données (RDF, RDFS, RDFa) et de faire des requêtes sur les données (SPARQL).

J'ai pu approfondir ces connaissances en participant à un MOOC donné par l'INRIA sur ce sujet. J'ai pu appliqué ces différents concept dans la réalisation de mon Portfolio.
