Projets Informatique

Projet WEB : Réalisation d'un site qui donne des avis d'expert sur des films

Projet en binôme avec Benjamin Harmand

Description : Le site permet aux internautes de consulter les commentaires d'expert sur divers films. Les experts ont des catégories préférées, un internautes peut suivre un expert et aura ainsi accès dans sa page personnel à tous les commentaires des experts qu'il suit. L'internaute peut faire des recherches par film et par expert. Il aura accès aux descriptions des films et ainsi qu'à la fiche de description des experts

Technologies : XHTML Strict 1.0, CSS 2, Javascript, PHP 5.3, base de données MySQL


Projet Java : Statistique et visualisation

Projet réalisé par groupe de quatre étudiants

Description : Le projet consiste à développer une application Desktop permettant de faire des calculs statistiques à partir d’un jeu de données et les représenter graphiquement. Il s'agit de réaliser une interface intégrant les fonctionnalités suivantes :

  • Un module permettant d’effectuer des calculs Statistiques
  • une lecture de fichiers normalisés avec gestion des données manquantes
  • un module de visualisation permettant de faire une représentation graphique des calculs statistiques sous forme d’histogrammes et de nuages de point

Technologies : Java, SWT (interface graphique), iText (export PDF), JFreeChart (graphique : histogramme, nuage de point)
